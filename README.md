# ЖуликGram (OSINTgram)

Шуточный симулятор специалиста управления собственной безопасности (СБ). Представляет собой "социальную сеть для жуликов", где нужно анализировать профили вымышленных кандидатов, искать компромат в их постах и комментариях, "пробивать" их логи через терминал и выносить вердикты — чист, подозрителен, опасен или требует немедленного задержания.

## 🚀 Как запустить проект локально

Проект разработан с использованием **React**, **Vite** и **Tailwind CSS**. Для локального запуска вам понадобится установленный на компьютере [Node.js](https://nodejs.org/).

### Шаги по установке и запуску:

1. **Скачайте или склонируйте проект** на свой компьютер.
2. Откройте терминал (командную строку) в папке с проектом.
3. **Установите зависимости**, выполнив команду:
   ```bash
   npm install
   ```
4. **Запустите сервер для разработки**:
   ```bash
   npm run dev
   ```
5. После успешного запуска в терминале появится ссылка (обычно это `http://localhost:5173/` или `http://localhost:3000/`). Откройте эту ссылку в любом современном браузере.

## 🛠 Технологии

- **React 19**
- **Vite** — быстрый сборщик
- **Tailwind CSS 4** — для быстрого и красивого оформления
- **Framer Motion** — для плавной анимации интерфейса (алерты, терминал)
- **Lucide React** — иконки

## 🎮 Как играть?
1. Введите любой позывной/имя сотрудника на стартовом экране.
2. Листайте ленту (Feed) и обращайте внимание на детали постов и комментарии недовольных коллег/родственников/жертв.
3. Заходите в профили "кандидатов".
4. Нажимайте кнопку **"Пробить по базам данных"**, чтобы запустить терминал OSINT-анализа и получить скрытую сводку.
5. Выносите вердикты в правой панели (на десктопе) или прямо в профиле (на мобильном).
6. Как только оцените всех кандидатов, нажмите **"Сдать отчет руководству"** и получите персональное поздравление!!!
